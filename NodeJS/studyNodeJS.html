기본 시작
  • 노드 JS 설치하기 
                    1. 설치 후 버전 확인 : node -v
                2. vscode의 터미널에서 npm install npx -g을 실행하여 npm 설치
오류발생
  • npm 업데이트를 위해 npm install -g npm할때 오류 발생 
                    1. 폴더의 보기에서 숨김 파일 보기 옵션 체크
                2. Node가 설치된 폴더에가서 npm과 npx에 임의의 이름 붙이기
                3. cmd를 관리자 권한으로 실행한 다음에 npm install -g npm 실행
